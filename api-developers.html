Notes

The API key should be sent as a Bearer token in the Authorization header of the request. Get your API key.

List

API endpoint:

https://seoai.com/api/vi/projects/(projects)

Request search

cur1 --location --request DELETE `https://seoai.com/api/projects/(project)` /
--header `Authorization: Bearer (api_key)`

Parameter

Type

Description

search

The search query

search_by

Search by. Possible values are: projects for Name . Defaults to: projects.

sort_by

Sort by. Possible values are: created_at for Date created, project for Name. Defaults to:created_at.

sort

Sort.Possible values are: desc for Descending, asc for Ascending. Defaults to: desc.

per_page

Results per page. Passible values are: 10, 25, 50, 100. Defaults to: 25.
